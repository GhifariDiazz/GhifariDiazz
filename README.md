## 📖 About Me

I'm a self-taught web developer, currently exploring both frontend and backend development. My journey started with simple HTML pages and has grown into building complete, functional web applications — including a portfolio site and an administration system used to manage real resident data.

I care most about:
- Writing clean, readable code as I learn
- Turning small ideas into working prototypes
- Getting better one project at a time, not all at once

<br/>
